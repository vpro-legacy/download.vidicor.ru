Инструкция по просмотру видеотрансляций «Vidicor Video System» (www.vidicor.ru)

Для просмотра видеотрансляций необходимо открыть интернет-браузер (напр., MS Internet Explorer, Mozilla Firefox, Apple Safari, Opera, Google Chrome) и зайти на указанную для этой трансляции веб-страницу (напр., http://live.vidicor.ru).

Если вы просматриваете трансляцию «Vidicor Video System» в первый раз, то на экране появится сообщение о том, что необходимо установить программу-плеер (плагин).

Ниже описан процесс установки для браузера MS Internet Explorer. В остальных браузерах процесс установки отличается незначительно. Для удобства на странице выдаются подсказки по шагам установки, свои для каждого из браузеров.

Шаг №1. Нажмите кнопку «Установить»:

Шаг №2. Система предложит скачать файл. Нажмите кнопку «Run» – «Запустить»:

Шаг №3. Установка плагина. Согласитесь с установкой плагина, нажимая «Далее» в мастере установки. Плагин Vidicor имеет сертификат международной службы VeriSign на его безопасность.

Шаг №4. Обновление страницы. Для MS Internet Explorer нажмите «F5» или ссылку «здесь». Остальные браузеры требуют закрытия страницы просмотра и открытия страницы просмотра вновь.

Установка плеера осуществляется один раз, в дальнейшем вы можете смотреть любые трансляции.

Если в ходе установки на компьютере появится сообщение о невозможности установки программы, это значит, что на этом компьютере для вас отключены права для установки программ. Обратитесь за помощью к системному администратору. Если окно программы появилось, а видео нет – убедитесь, что всё нормально с интернет-подключением.

Для качественного просмотра желательно, чтобы скорость Вашего подключения к Интернету была не менее 512 кбит/с.

Общая информация системе «Vidicor Video System» представлена на сайте разработчика www.vidicor.ru.
